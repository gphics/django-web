# Finguard

### Description
A web solution that help detects patterns and anomalies in the spending volume of a user and also gives a comprehensive descriptive statistics.

### Features
    1. Anomaly detections
    2. Summary statistics 
    3. Clustering users base on daily spending

### Models
    1. User (Default)
    2. Profile (spending_class, user_id,  coutry, contact, state, city)
    3. Category (title)
    4. Transactions (amount, user_id , category, datetime, flagged, user_verfied)
